Задача: Написать программу, которая из имеющегося массива
строк формирует новый массив из элементов, длина которых меньше,
либо равна 3 символам. Первоначальный массив можно ввести с
клавиатуры, либо задать на старте выполнения алгоритма.
лучше обойтись исключительно массивами.

Решение: 
Сздадим массива с размером от 3 - 5. Предоставим пользователю заполнить его
произвольными строковыми значениями. По мере заполнения массива считаем
сколько строковых данных было размером меньше или равно 3 ед длины. 
Результат счета будет являтся размер нового массива, это сделано для того,
чтобы массив не заполнянлся пустыми строками.
Далее, на основе имеющегося массива созадим новый из старого, для этого
создадим цикл в котором он снова будет проходить по элементам первого массива
и проверять длину всех элементов, далее при выполнении условия в новый массив запишется
значение первого массива по индексу.
В результате получим массив, чьи элементы удовлетворяют условию задачи.
